# bd_housing_property_analysis
Here I scraped data from the first page of (https://www.bdhousing.com) and cleaned and analyzed the raw data
First I used (https://webscraper.io/) google chrome extention to scrap the data from the landing page of the bdhousing website 
After scraping the data I saved it in CSV formet than imported it to powerbi where I used power query to clean and modify the data 
After that I tried to find out what information the data is giving and tried to visualize them.
The data volume is very low and also it is too much messy as you can check it at the given CSV file to see the cleaned data and the visualization please 
download the pbix file and open it in you device's Power Bi application.
